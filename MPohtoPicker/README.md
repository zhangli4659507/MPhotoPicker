## 系统要求
* iOS7.0+
* Xcode6.0.1+

## 核心类名
* MBigImageView.h 处理图片的放大缩小等
* MAllPtotoIma.h  从相册中获取相册图片
* #import "MShowAllIma.h" 显示一个相册中所有缩略图片
* MBigImaCollectionView.h 处理大图浏览
